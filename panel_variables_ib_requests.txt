ib_client_bis methods -> variables shown in panels

- is_connected:
-> ib_connected (Status Panel) : {True, False}

- managed_accounts:
-> account (Status Panel) : DU123456 or DU654321 or --

- req_current_time:
-> server_time_text (Status Panel) : 15:30:45 or --
-> latency_text (Status Panel) : 25 ms or --

- account_summary:
-> net_liquidation (Portfolio Panel) : 100000 USD or --
-> total_cash_value (Portfolio Panel) : 25000 USD or --
-> available_funds (Portfolio Panel) : 24000 USD or --
-> unrealized_pnl (Portfolio Panel) : 120.5 USD or --
-> realized_pnl (Portfolio Panel) : 80.0 USD or --
-> gross_position_value (Portfolio Panel) : 55000 USD or --

- positions:
-> position_symbol (Portfolio Panel) : EURUSD or AAPL or ?
-> position_qty (Portfolio Panel) : 1 or 0 or -2

- ticker:
-> bid (Chart Panel) : 1.08421 or None
-> ask (Chart Panel) : 1.08423 or None
-> last (Chart Panel) : 1.08422 or None
-> close (Chart Panel) : 1.08390 or None
-> volume (Chart Panel) : 12345 or 0 or None
-> market_time (Chart Panel) : 2026-02-15 14:45:10 or None

- open_trades:
-> open_order_id (Orders Panel) : 12345 or 67890
-> open_order_symbol (Orders Panel) : EURUSD or AAPL or ?
-> open_order_side (Orders Panel) : {BUY, SELL}
-> open_order_qty (Orders Panel) : 10000 or 1 or 0
-> open_order_price (Orders Panel) : 1.0842 or 185.5 or None
-> open_order_status (Orders Panel) : {PendingSubmit, PreSubmitted, Submitted, Filled, Cancelled, Inactive}

- fills:
-> fill_time (Orders Panel) : 2026-02-15 14:45:10 or other timestamp
-> fill_symbol (Orders Panel) : EURUSD or AAPL or ?
-> fill_side (Orders Panel) : {BOT, SLD, BUY, SELL}
-> fill_qty (Orders Panel) : 10000 or 1 or other quantity
-> fill_price (Orders Panel) : 1.0842 or 185.5 or other price
-> ib_event_time (Logs Panel, fill) : 2026-02-15 14:45:10 or other timestamp
-> ib_event_message (Logs Panel, fill) : Order 12345 filled or other fill message
-> ib_event_type (Logs Panel, fill) : {fill}

- req_pnl:
-> daily_pnl (Performance Panel) : 12.3 or 0.0 or nan
-> unrealized_pnl (Performance Panel) : 4.5 or 0.0 or nan
-> realized_pnl (Performance Panel) : 7.8 or 0.0 or nan
-> pnl_daily (Risk Panel) : 12.3 or 0.0 or nan
-> pnl_unrealized (Risk Panel) : 4.5 or 0.0 or nan
-> pnl_realized (Risk Panel) : 7.8 or 0.0 or nan

- req_pnl_single:
-> pnl_single_position (Performance Panel) : 1 or 0 or -2

- req_executions:
-> executions_count (Performance Panel) : 0 or 42 or other integer

- account_values:
-> account_value_tag (Risk Panel) : NetLiquidation or AvailableFunds or other IB account tag
-> account_value (Risk Panel) : 100000 or 100000.0 or other value

- what_if_order:
-> what_if_init_margin_change (Risk Panel) : 100 or 100.0 or other value
-> what_if_maint_margin_change (Risk Panel) : 80 or 80.0 or other value
-> what_if_commission (Risk Panel) : 1.50 or 1.5 or other value

- trades:
-> ib_event_time (Logs Panel, order_status) : 2026-02-15 14:45:10 or other timestamp
-> ib_event_message (Logs Panel, order_status) : Order 12345 status Submitted or other status message
-> ib_event_type (Logs Panel, order_status) : {order_status}

- executions:
-> ib_event_time (Logs Panel, execution) : 20260215 14:45:10 or other timestamp
-> ib_event_message (Logs Panel, execution) : Execution 12345 for EURUSD or other execution message
-> ib_event_type (Logs Panel, execution) : {execution}

Notes
- req_mkt_data is required to start chart streaming, but chart values are read from ticker.
- ib_event_type = "error" does not come from these polling methods; it comes from IB error callbacks/events.
